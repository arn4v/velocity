# Velocity

Velocity is my Next.js-based project boilerplate with all the libraries I frequently use to make my side-projects.

## Stack
1. Next.js
2. TypeScript
3. TailwindCSS
4. Auth0 + Auth0's Next.js SDK
5. Prisma
6. Postgres/MySQL
